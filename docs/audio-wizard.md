# Audio & Music Wizard

Guided step-by-step workflow for creating audio CDs, copying music to disc, and ripping audio CDs.

---

## Create Audio CD

Burn audio files to a standard Red Book Audio CD playable in any CD player.

### Supported Audio Formats

| Format | Description |
|:-------|:------------|
| WAV | Uncompressed PCM audio |
| MP3 | MPEG-1 Audio Layer 3 |
| FLAC | Free Lossless Audio Codec |
| AAC / M4A | Advanced Audio Coding |
| OGG Vorbis | Open-source compressed audio |
| WMA | Windows Media Audio |
| APE | Monkey's Audio (lossless) |
| WV | WavPack (lossless) |

### Audio CD Options

| Option | Description |
|:-------|:------------|
| **Track Gap** | Silence between tracks (0–10 seconds, default 2) |
| **Normalization** | Equalize volume across all tracks |
| **CD-Text** | Write artist, title, and track name metadata |
| **Gapless Playback** | Remove gaps between tracks |
| **Write Mode** | SAO (Session At Once) recommended for audio CDs |
| **Write Speed** | Typically 4x–8x for best audio quality |

---

## Rip Audio CD

Extract audio tracks from an existing CD to digital files.

### Ripping Options

| Option | Description |
|:-------|:------------|
| **Output Format** | WAV, MP3, FLAC, AAC, OGG |
| **Paranoia Mode** | Enable multi-pass error correction for damaged discs |
| **Jitter Correction** | Correct timing jitter during extraction |
| **Track Selection** | Choose specific tracks or rip entire disc |

---

## Copy Music Disc

Copy an audio CD disc-to-disc or to an image file for later burning.
