# Data Disc Wizard

Guided step-by-step workflow for creating and burning data discs from files and folders.

---

## Workflow

1. **Select drive** — Choose the target optical drive
2. **Add files** — Select files and folders from your computer
3. **Configure filesystem** — Choose ISO 9660, Joliet, UDF, or hybrid
4. **Set options** — Volume label, write speed, verify after burn
5. **Burn** — Write the data disc

The wizard shows capacity usage and warns if data exceeds disc capacity.

---

## Filesystem Options

| Option | Description |
|:-------|:------------|
| **File System** | ISO 9660, ISO 9660 + Joliet, ISO 9660 + UDF, UDF, Rock Ridge, HFS+ |
| **Volume Label** | Disc name shown when mounted |
| **Allow Long File Names** | Beyond ISO 9660 8.3 limit |
| **Allow Deep Directories** | Below 8 directory levels |
